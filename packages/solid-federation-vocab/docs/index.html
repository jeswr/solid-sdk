Domain ontologies — the gUFO Core + sectors

A foundational Solid Core ontology (re-based onto gUFO, the gentle OWL-2-DL implementation of the Unified Foundational Ontology) and the sector ontologies that import and constrain it. These are the IRIs an app's fedapp:sector points at (…/sectors/<sector>#sector). Each reuses real vocabularies (FIBO for finance, FHIR-mode-A + QUDT for health, vCard for contacts, schema.org for media, iCalendar for scheduling, eIDAS/ISO for identity) and passes a HermiT consistency gate (0 unsatisfiable classes).
