# Generated bookmarks-sector model (PoC)

Generated from `bookmarks.ttl` and `bookmarks.shacl.ttl` by
`@jeswr/federation-codegen`. The output is a structural projection of the SHACL graph,
not a security-reviewed application model.

The accessors enforce RDF term projections and setter cardinality through
`@rdfjs/wrapper`. Consumers still need SHACL validation, domain-specific input checks,
fail-closed builders, and read-path hardening. Do not publish this proof-of-concept package.
