# Federation codegen PoC

Review-only generator that projects a sector ontology and SHACL profile into a private typed model
package. It emits vocabulary constants and `@rdfjs/wrapper` accessors for the supported structural
subset; it does not use LDO or infer application/security policy.

Generate the bookmarks proof:

```sh
pnpm --filter @jeswr/federation-codegen run generate:bookmarks
```

Gate the tool and the generated package:

```sh
pnpm --filter @jeswr/federation-codegen run gate
```

The generated bytes are committed at `generated/bookmarks-sector`; a test regenerates the package
in a temporary directory and compares every output. The behavior tests use `rdf-validate-shacl` to
prove conforming accessors and to expose the sector profile's missing HTTP(S) URL policy.

See the [PoC design note](../../docs/design/federation-codegen-poc.md) for the
generated-versus-hand-written comparison, open design questions, and the no-go replacement decision.
